Location-Based Access Control

Description

This project implements a location-based access control system using Laravel, MySQL, and JavaScript. The system restricts access to registered users, allowing them to log in only when they are within 100 meters of the company's location. The submit button is disabled for users outside this range.

Features
Users can log in only if their current location is within a 100-meter radius of the company. Registration Page: Users can register with the system. Login Page: Users can log in when their location is within the specified range. Dashboard: Displays relevant information or functionalities for logged-in users.
